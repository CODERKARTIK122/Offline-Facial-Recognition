# Offline-Facial-Recognition

Offline Facial Recognition & Liveness Detection System

Overview

A mobile-based offline authentication system designed for remote locations with limited or no internet connectivity. The solution performs facial recognition and liveness detection directly on the device and synchronizes records with cloud infrastructure once connectivity is restored.

Features

- Offline Facial Recognition
- Blink-Based Liveness Detection
- React Native Cross-Platform Support
- Local SQLite Storage
- Offline Authentication Workflow
- Future AWS Sync Capability
- Lightweight Mobile-Optimized Architecture

Technology Stack

- React Native
- TensorFlow Lite
- MobileFaceNet
- SQLite
- AWS Cloud Integration

Application Workflow

1. User Login
2. Face Capture
3. Liveness Verification
4. Face Matching
5. Authentication Result
6. Offline Data Storage
7. Cloud Synchronization
